# codex-practice

A mobile-friendly programming study roadmap site.

## Open locally

Install dependencies and run:

`npm install`

`npm start`

Then open:

`http://localhost:3000`

## Publish for phone use

This repository includes a GitHub Pages workflow. After GitHub Pages is enabled for this repository, the site can be published from GitHub Actions.

Expected URL:

`https://tbox-creater.github.io/codex-practice/`
