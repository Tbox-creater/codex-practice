Operating System

操作系统专题

把进程线程、内存、文件 IO、调度和并发问题串起来学，让你更理解程序到底是怎么跑起来的。

Core

基础认知主线

进程与线程

  • 进程资源隔离与线程共享资源。
  • 上下文切换为什么贵。
  • 多进程和多线程适用场景。

内存管理

  • 堆与栈、虚拟内存、分页。
  • 内存泄漏、内存碎片和大对象问题。
  • 程序为什么会 OOM。

文件与 IO

  • 阻塞 IO、非阻塞 IO、同步异步。
  • 文件描述符和网络 socket 的关系。
  • 为什么高并发服务要关注 IO 模型。

调度与并发

  • 时间片、优先级、抢占式调度。
  • 竞争条件、死锁、饥饿与活锁。
  • 锁、信号量和条件变量概念。

Applied

结合开发理解系统

Web 服务

为什么连接数高时会受线程模型和 IO 模型影响。

数据库

为什么缓存命中、页大小和磁盘 IO 会影响 SQL 性能。

前端

为什么浏览器多进程、多线程设计会影响渲染和脚本执行。

排障

CPU 飙高、内存泄漏、文件句柄耗尽时分别先看什么。

Interview

操作系统高频问题

进程和线程的区别是什么？

不要只答“线程更轻量”，要能讲资源、切换成本和通信方式。

为什么会发生死锁？

要能说清必要条件和常见规避策略。

同步和异步、阻塞和非阻塞怎么区分？

这是很多人会混的点，面试里很容易被追问。

一个程序占用内存越来越高，你怎么排查？

从对象增长、缓存、循环引用和资源未释放来展开更有说服力。
